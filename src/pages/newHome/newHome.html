Hi, my name is Wellington and I'm a computer scientist from Brazil, currently working as a Front-End developer.

I like making software that makes me more productive and people's lives easier.

I'm a Linux enthusiast with a passion for music and weightlifting.

Here are some of my projects:

powerlifting app (soon)

A workout management app built with React Native with the goal of improving readability mid training session of training programs that rely on the use of big and complex spreadsheets, of sports such as powerlifting and weightlifting. It currently features a editor and provides a RM calculator and a weight plate calculator.

wmUI

A responsive website that I made out of boredom with the intention to use it as a startup page for my browser, that resembles a Linux desktop window manager interface.

OpenCV.js Barbell Tracker

This is my university gradutaion project. A JavaScript application for weightlifting video analysis that runs the OpenCV library directly in the browser, with the following functionalities:

- Barbell Path Tracker - Realtime Concentric Velocity - Realtime Acceleration - Realtime Horizontal Displacement
