Hi, my name is Wellington and I'm a computer scientist from Brazil, currently working as a Front-End developer.

I like making software that makes me more productive and people's lives easier.

I'm a Linux enthusiast, obsessed with performance, and with a passion for music and weightlifting.

Here are some of my projects:

BarbellWhip

An open source workout management app built with React Native, with the goal of improving readability of complex training programs that make use of big spreadsheets mid training session, of sports such as powerlifting and weightlifting. It currently features a program review page, a program editor and a RM(repetition max) and weight plate calculators.

papibaquigrafo

A CLI tool for personal use written in Go to download manga from various sites.

wmUI

A responsive website that I made out of boredom with the intention to use it as my browser's startup page, that resembles a Linux Window Manager interface.

OpenCV.js Barbell Tracker

This is my university gradutaion project. A JavaScript application for weightlifting video analysis that runs the OpenCV library directly in the browser, with the following functionalities:

  • Barbell Path Tracker
  • Realtime Concentric Velocity
  • Realtime Acceleration
  • Realtime Horizontal Displacement
